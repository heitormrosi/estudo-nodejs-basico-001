# Estudo sobre NodeJS

## Implementações a fim de estudo dos livros "Primeiros passos com Node.js", de João Rubens, e "Aprendendo Node", de Shelley Powers.

## Descrição dos livros

> Os livros "Aprendendo Node" e "Primeiros passos com Node.js" orientam o desenvolvedor nos estágios iniciais, abrangendo desde a instalação do ambiente em múltiplas arquiteturas, como Windows, Linux e Raspberry Pi, até o uso de ferramentas de suporte como o editor Atom e o terminal Cmder. Antes da codificação, as obras detalham a base teórica da plataforma, explicando o papel fundamental da engine V8, o funcionamento monothread e a natureza assíncrona orientada a eventos do Node.js, que utiliza o Event Loop para gerenciar milhares de requisições de forma não bloqueante. O aprendizado prático se inicia com a criação de um servidor web básico por meio do módulo nativo http, ensinando a manipular os objetos de solicitação (request) e resposta (response) para retornar conteúdos em HTML ou JSON. A organização do sistema é abordada através do sistema de módulos CommonJS, em que se aprende a exportar funcionalidades e gerenciar dependências com o NPM, centralizando as configurações do projeto no arquivo package.json. Por fim, para otimizar a produtividade inicial, os livros destacam o uso do console interativo REPL para testes rápidos e a instalação do módulo Nodemon, que reinicia automaticamente o servidor a cada alteração salva no código.
